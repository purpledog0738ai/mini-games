# Git 커밋 규칙

## 커밋 메시지 형식

```
[TAG] 제목

- 변경사항 1
- 변경사항 2
```

## 태그 종류

| 태그 | 용도 | 예시 |
|------|------|------|
| `ADD` | 새 기능/파일 추가 | [ADD] 핑퐁 게임 추가 |
| `FIX` | 버그 수정 | [FIX] 터치 이벤트 안 되는 문제 |
| `UPDATE` | 기존 기능 수정/개선 | [UPDATE] 캐릭터 디자인 변경 |
| `REMOVE` | 삭제 | [REMOVE] 사용 안 하는 파일 정리 |
| `REFACTOR` | 코드 구조 개선 | [REFACTOR] 게임 로직 분리 |
| `STYLE` | UI/디자인 변경 | [STYLE] 버튼 색상 변경 |
| `DOCS` | 문서 작성/수정 | [DOCS] README 추가 |
| `CHORE` | 기타 (설정, 빌드 등) | [CHORE] .gitignore 수정 |

## 예시

```
[ADD] 2048, 핑퐁 게임 추가

- 2048 숫자 퍼즐 게임
- 핑퐁 CPU 대전 게임
- 캐릭터 SVG 2개 추가
```

```
[FIX] 모바일 스와이프 버그 수정

- 게임 영역에서 스와이프 안 되던 문제 해결
- 터치 이벤트 오버레이 추가
```

```
[UPDATE] 시작 화면 UI 개선

- 전체화면 오버레이로 변경
- 캐릭터 썸네일 추가
```

## 규칙

1. 태그는 대문자로
2. 제목은 간결하게 (50자 이내)
3. 내용은 하이픈(-)으로 시작
4. 한글 사용 OK
